开关效果--honeySwitch

引入honeySwitch.css和honeySwitch.js

仿手机端的一个开关按钮，插件在PC和手机端都是用，不支持IE8，在IE8中做了单独处理，会展示一个边框

在原有的基础上，做了改变，添加了可以写进文字的功能。

honeySwitch的使用--基本

switch-off表示默认关闭，switch-on表示默认开启

honeySwitch的使用--禁用

添加class:switch-disabled.按钮被禁用

honeySwitch的使用--改变颜色大小

1.默认主题色为绿色,直接使用js的方式（honeySwitch.themeColor="颜色值";）修改默认主题色。

2.定制单个元素主题色，给元素添加themeColor="颜色值"属性

3.定制单个元素大小，给元素添加width="80"属性

honeySwitch的使用--回调

switchEvent("#id", function () {}, function () {});

honeySwitch扩展--添加文字

1、haveword="true"属性，代表元素是否含有文字

2、openword=""属性，是开启状态显示的文字

3、closeword=""属性，是关闭状态显示的文字
